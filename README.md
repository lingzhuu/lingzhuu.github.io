# Lingzhu's Personal Website

This is my personal website built with [Quarto](https://quarto.org/) and hosted on [GitHub Pages](https://pages.github.com/).

## Live Website

The website is hosted at: [lingzhuu.github.io](https://lingzhuu.github.io/)

> **Note**: The `develop` branch contains updates under development, including publications and other sections that are not yet live on the main website.

---

## Project Structure

- **`main`**: Contains the current live version of the website.
- **`develop`**: Used for updating the website in the future.

Key files and directories:
- `_quarto.yml`: Global configurations for the website.
- `index.qmd`: The main homepage of the website.
- `publications.qmd`: List of publications (to be updated).
- `files/`: Directory for storing images and other files used in the website.
